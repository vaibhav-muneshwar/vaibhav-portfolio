Engineering Philosophy

Moving beyond simulation to build deployable, silicon-level solutions.

I am an R&D System Engineer obsessed with the intersection of low-level hardware and intelligent firmware. My work is rooted in the belief that software is only as robust as the silicon it runs on.

Whether it is resolving strict RGB565 bit-alignment for parallel TFT displays, securing IoT nodes with hardware-accelerated AES-256, or deploying edge AI for agricultural diagnostics, I prioritize deterministic performance, memory safety, and real-world reliability.

I don't just write code; I synchronize it with the physical world using logic analyzers, oscilloscopes, and precise register-level debugging.
